# FragGeneScan++
An improved version of [FragGeneScan-Plus][fgsp].

# Build
To build FragGeneScan++ from source, run the following commands from the source
repository:

```sh
make clean
make
```

To run FragGeneScan++, see the [USAGE.md](USAGE.md) file.
